核心法則

從食物獲得的能量，必須超過獲取食物時所消耗的能量——否則任何採集行為都將是徒勞。

01 // FOOD SAFETY

能量平衡與可食用性測試

採集前先評估投入與回報比，再依序測試植物安全性。

🌿 植物資源

相對容易大量獲取，但必須通過嚴格的可食用性測試。未知蕈類切勿嘗試，辨識錯誤的代價極高。

🐟 水生資源

魚類含有豐富蛋白質，透過定置被動捕撈方式，能以最低體能消耗獲得高熱量補充。

⚗️ 通用植物可食用性測試

測試期間禁食其他食物。任何步驟出現燒灼、刺痛或紅腫，立即停止並以大量清水沖洗。

S1
皮膚測試 — 等待 15 分鐘

將植物汁液擦拭於手肘內側，觀察有無發炎、發癢或紅腫。

S2
唇部測試 — 等待 15 分鐘

若無皮膚過敏，取少量植物貼在嘴唇外側，檢視有無刺痛感。

S3
舌尖口腔測試 — 等待 15 分鐘

嚼碎後含於舌尖，切勿吞下。若有強烈苦味、肥皂味或刺痛，立刻吐掉。

S4
少量吞嚥測試 — 空腹觀察 8 小時

吞下極少量，嚴格空腹觀察。若有胃痛、噁心，催吐並大量飲水。

S5
小量確認 — 再觀察 8 小時

煮食約半杯同款植物，完全無恙後，方可將該部位視為安全食材。

02 // COOKING

現地低資源烹飪安全

缺乏金屬鍋具時，利用自然媒材進行高溫熟食處理，消滅寄生蟲與有害細菌。

🏺 泥土陶烘法

用無毒大型葉片（香蕉葉、月桃葉）完整包裹食材，外部均勻糊上約 2.5 cm 厚 的乾淨陶土，直接放入炭火灰燼中烘烤 30～60 分鐘。敲開泥殼時，鱗片與外皮會隨之脫落，可獲得乾淨熟透的食物。

🪨 石頭導熱烘烤

堆疊表面平滑的大型石頭，升火徹底加熱後清除炭火，將食材鋪在滾燙岩石上烘烤。

⚠ 安全警示 絕對避免使用溪床孔洞岩石——內部水分受熱膨脹將導致岩石劇烈爆裂。

03 // FISHING

簡易漁具製作與漁獲衛生處理

點擊卡片查看利用隨身物品製作被動捕撈工具的方法。

📌 別針 / 釘子改製魚鉤 點擊查看製作方式 →

用工具將安全別針或硬質釘子折彎，以石頭磨尖末端。雖無倒鉤，但配合繃緊線路或彈力枝條，可達被動定置捕撈效果。

🪵 天然木頭 / 骨頭夾鉤 點擊查看製作方式 →

削尖約 2.5 cm 的硬質雙頭木針，將線綁在正中央並平行貼附。魚吞食後線抽緊，木針橫向卡入魚體內部（橫軸夾鉤法）。

🚨 緊急醫療 — 魚鉤刺入人體處理

若魚鉤刺入且倒鉤已深入，千萬不可直接往後硬拉。正確做法：順著刺入方向將魚鉤向前推，讓倒鉤穿透外露，用剪鉗剪斷倒鉤後再順向退出，立即消毒包紮。

⚠️ 寄生蟲防線 — 淡水魚絕對不可生食

淡水魚內部常帶有顎口線蟲、中華肝吸蟲等致命寄生蟲，必須徹底加熱熟透方可食用。

F1
徹底放血與去黏液

切斷魚鰓基部放血，以清水或乾淨細沙揉搓魚體，去除帶有細菌的表皮黏液。

F2
剖腹與摘除內臟

從肛門沿腹部中線向前剪開。絕對不可弄破魚膽——膽汁外溢會使整批肉品遭受污染，若不慎破裂須立刻以大量清水沖洗。

F3
去皮與去骨加工

無鱗魚（如鯰魚）於胸鰭後方劃切口，用布包住外皮向下撕除。較大魚體可沿脊椎骨兩側切出兩片無刺魚排 (Fillet)，利於快速高溫烤熟。

04 // MAMMALS

哺乳類被動獲取與衛生處理

🚨 雙重安全警告
  • 兔肉中毒症 (Rabbit Starvation)： 小型哺乳類肌肉幾乎不含脂肪，連續數天幾乎只依賴瘦肉蛋白時會引發急性蛋白質中毒（噁心、腹瀉、虛弱）。短期食用（1–3 天）不構成立即威脅——求生情境下仍應積極取食。真正的解方是搭配任何脂肪來源（昆蟲、堅果、魚）即可大幅降低風險，而非拒絕食用。
  • 傳染病防範： 野生動物體表可能帶有兔熱病等桿菌。手部有傷口時，絕對不可赤手進行剝皮與清理內臟。

🔧 現地延伸防護工具

🥢 雙叉結構杆

末端具自然分叉的硬質樹枝削尖叉頭，用於涉水或探查洞穴時的防禦與定位限制。

🎯 彈力延伸觀測具

Y 字型樹枝搭配彈性膠條，低耗能獲取高處落果或擊落小型飛禽。

🪶 繩套延伸杆

長枝條末端固定可自由抽緊的繩環，遠距離套取移動遲緩生物，降低直接接觸風險。

小型哺乳類剝皮與處理流程

M1
開口與皮肉分離

於腹部中央橫向割開一道切口，用手指伸入小心將皮膚與肌肉層分離，逐步向兩側擴展。

M2
手套式整體剝皮 (Glove Skinning)

切斷前後肢關節，一手按住肌肉，另一手抓住腹部外皮，像脫手套一樣順向往後拉除整張外皮。確保毛髮完全不接觸內部肉質。

M3
摘除內臟與清洗

刀尖朝上，從恥骨向胸腔縱向剪開腹壁，避免刺破腸道。拉出內臟後，若保留心肝食用，須確認表面無異常白點或病變，並以清水徹底沖洗腹腔。

⚠ 大型哺乳類處理要點
  • 即刻放血： 頭部朝下置於斜坡，於頸動脈下刀深刺放血，顯著延緩肉質腐敗速度。
  • 防污染切線： 刀尖始終朝外、鈍端朝內。嚴格禁止刺破胃囊或腸道，消化物外溢將導致整塊肉品細菌污染報廢。
  • 部位肢解： 依後腿、前腿、背脊里肌分塊切卸後，懸掛於通風陰涼處，利用表面自然風乾形成的微乾燥外膜阻絕昆蟲產卵。
  • 皮毛乾燥： 刮除內側殘餘脂肪，用枝條製作方形框架將皮毛四周拉平撐開，置於通風處乾燥。
05 // INSECTS

低能耗昆蟲資源採集

昆蟲含高密度蛋白質與微量元素，且採集時體能消耗極低。主動避開體表多毛或色彩鮮豔的個體——此類特徵通常暗示帶有防禦性毒素。

🪵 防水布朽木震動採集法

在地面鋪設乾淨防水布，將潮濕朽木移至中央輕輕敲碎。大量藏匿於內部的幼蟲與蟻類受驚落於防水布上，可快速篩選採集，比徒手挖掘效率高出許多。

🌙 夜間垂直光照收集法

在兩樹間垂直張開白色衣物，下方放置裝滿清水的淺容器。夜間以手電筒照射布幕，具趨光性的夜行昆蟲飛往布幕後跌落水盆，可高效收集且不易逸散。

06 // REPTILES

兩棲爬蟲安全處置

🚨 無脊椎體內毒素排空
  • 蝸牛、蛞蝓與螺類體內常殘留植物毒素，食用前須留置飢餓 24 小時或以確認安全的植物餵食排空，且必須徹底煮熟。
  • 切勿食用海生蝸牛。

🐍 蛇類安全處理三步驟

R1
安全去頭

切除頭部後，必須原地靜置至少 15 分鐘，防範死後神經反射引發的意外咬傷。

R2
剖腹清理

用刀尖由前向後小心劃開腹壁，完整摘除內部臟器與消化道。

R3
套袖剝皮

於頸部後方切開皮層，一手拉住肉體，另一手由尾部向頭部方向將整張外皮成套撕下。

07 // AQUATIC

溪谷甲殼類與貝類毒素防禦

🚨 有毒藻類（紅潮）威脅
  • 夏季或鄰近海口區域時，高度避免採集水生貝類。赤潮引發的藻類毒素可導致心悸、嘔吐、腹瀉、發燒等嚴重中毒症狀。

🐚 淡水貝類烹煮規範

開殼前以乾淨水源洗淨。持續加熱至外殼因高溫完全張開方可食用，煮後仍緊閉者必須丟棄。

🦐 甲殼類（蝦蟹）烹煮規範

在淡水中持續煮沸不低於 15～20 分鐘。大型個體建議用竹竿串起進行長時間炭火慢烤，確保核心完全熟透。

08 // BIRDS

禽類被動留置與衛生處理

主動追捕禽類體能消耗過大，應利用其築巢、覓食與水源路徑進行被動置留。

📐 4 形重力陷阱

三根刻有精準凹槽卡榫的木條互相支撐，上方壓置重石板。觸動機關時卡榫解開帶動重物下壓，結構簡單且不損壞目標物。

🌿 樹枝連續圈套

在禽類棲息的橫向樹幹上，以細金屬線製作一整排直徑約 2.5～5 cm 的連續微型圈套，禽類沿枝走動時雙腳自然落入被動卡鎖。

🌾 水岸樹籬陷阱

於水鳥頻繁活動的淺水岸搭建低矮阻隔樹籬，中央留下數個通道並設置圈套，利用水鳥順向游動的習性進行被動置留。

🪶 去毛加工方式選擇

直接去皮法： 速度最快，但會流失富含高熱量的皮下脂肪層。

標準拔毛法： 先浸入熱水數分鐘使毛孔擴張，再順著羽毛的反方向拔除，保留完整皮層脂肪與營養價值。

🫁 內臟摘除要點

由下腹部縱向小心切開，完整拉出內臟管線。切勿刺破消化道與泄殖腔，防止糞便細菌污染肉質。
