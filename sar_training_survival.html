S2 · 技術執行 // GEAR ARCHITECTURE

個人生存裝備
三線配置檢查

3-Line Survival Kit Compartmentalization。任務出發前執行個人自主裝備盤點。第一線與第二線裝備在遭遇山難被迫「丟包（捨棄大背包）」時，必須完全留在身上——這是你原地存活的最低底線。

FIRST LINE · 必須固定於身上，任何情況不得脫離
戰術刀具 固定刃或折疊刀，含刀鞘或鎖扣。刀刃長度 7–12cm 為山域最佳。
L1-必備
點火系統（二合一） 打火石 + 防水火柴，分開存放。至少一套完全防水。
L1-必備
個人求生金屬盒（PSK） 含水泡貼 × 4、淨水藥錠 × 8、哨子、求生毯、備用藥（個人用藥）、便條紙與原子筆。
L1-必備
緊急保暖層（隨身） Buff / 毛帽 / 薄手套。防止失溫的第一道防線，必須穿戴在外套內層隨時可取得。
L1-必備
小型 LED 頭燈（隨身備份） 最小型號，裝入口袋。不取代 L2 主頭燈，僅作零件失效備份。
L1-建議
注意事項
丟包原則：第一線裝備必須與衣物牢固整合。山難被迫拋棄大背包時，這五樣物品是讓你在 72 小時內存活的最低配置。建議縫製或使用掛載系統固定在外套上。

SECOND LINE · 行動安全層 — 裝備於腰包 / 外掛系統
完整急救包（IFAK） 止血帶、彈性繃帶、消毒棉片、三角巾、醫療膠布、個人處方藥。
L2-必備
主頭燈 + 備用電池 亮度 ≥ 200 流明，電池額外攜帶一組。不使用充電式電池（低溫衰退嚴重）。
L2-必備
個人水袋 / 水瓶（2L） 當日用水量基準。配合 PSK 淨水藥錠可處理野外水源。
L2-必備
反光信號板 / 鏡 日間定向搜尋時向搜救直升機或搜救隊反光定位。有效距離可達 10km。
L2-必備
地圖 + 指北針 1:25000 或 1:50000 紙本地形圖（防水袋裝）+ 基礎指北針。不依賴電子設備。
L2-必備
通訊與導航裝備
備用通訊裝置 無線電（VHF/UHF）或衛星定位發報器（PLB / Garmin inReach）。行前確認電量 ≥ 80%。
L2-強建議
備用保暖衣物 輕量保暖中層（化纖或羽絨），壓縮裝入防水袋。備用手套一副。
L2-強建議
L2 特別提示：L2 裝備是「移動中仍可存活」的關鍵條件。若遭遇突發事故必須在山中過夜，L2 完整性決定你能否安全撐過第一個夜晚。建議裝入腰包或高掛點，不放在背包底層。
THIRD LINE · 基底營地層 — 主背包
外帳 / 露宿袋（Bivy Sack） 緊急避難所最低標準。防水係數 ≥ 3000mm。外帳優先考量架設速度（≤ 5 分鐘）。
L3-必備
睡眠系統（睡袋 + 睡墊） 溫度係數選擇：依預期最低溫再減 5°C。睡墊 R 值 ≥ 2.0（輕量任務）。
L3-必備
高熱量預備糧食（+1 日份） 攜帶超出計畫天數 1 日的緊急口糧。行動糧選擇高脂肪高碳水比例（堅果棒、壓縮餅乾）。
L3-必備
烹飪系統（爐具 + 鍋具） 高山氣罐爐具 + 鈦合金鍋。確認氣罐容量足夠行程天數。低溫環境使用四季罐。
L3-建議
大容量水處理（濾水器） 中空纖維濾心（Sawyer / MSR）。可配合 L1 淨水藥錠雙重防護野外水源。
L3-建議
⚠️ 丟包決策點：若必須拋棄 L3（主背包）執行緊急撤退，確認 L1 + L2 已完整穿戴後方可行動。不攜帶 L1/L2 拋棄 L3 是山難最常見的致命錯誤。

生理代謝與營養管理

POST-CONTACT METABOLIC & NUTRITION MANAGEMENT。長時間飢餓與脫水後的不當補給，可能造成比原本傷害更嚴重的醫療後果。此矩陣依據脫水程度與飢餓天數，產出分階段處置指引。

MODULE 03 · 受難者尋獲後生理評估 輸入受難者狀態，產出分階段處置矩陣
再餵食症候群（Refeeding Syndrome）警示：飢餓 > 5 天或嚴重脫水的受難者，在過快補給高熱量食物後，可能因電解質急劇失衡引發心律不整、橫紋肌溶解甚至死亡。第一原則永遠是「先水分、後電解質、再食物」，且必須緩慢分階段執行。
